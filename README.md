# zero.vim

### This plugin defines the following functions for manipulating text for search and substitute:

* zero#Trim()
* zero#CCword()
* zero#Cword()
* zero#Word()
* zero#Vword()
* zero#Pword()
* zero#grep#CCword()
* zero#grep#Cword()
* zero#grep#Word()
* zero#grep#Vword()
* zero#grep#Pword()
* zero#shell#CCword()
* zero#shell#Cword()
* zero#shell#Word()
* zero#shell#Vword()
* zero#shell#Pword()
* zero#substitute#CCword()
* zero#substitute#Cword()
* zero#substitute#Word()
* zero#substitute#Vword()
* zero#substitute#Pword()
* zero#InsertCCword()
* zero#InsertWord()
* zero#InsertPword()

### This plugin defines the following commands:

* ReplaceTypographicCharacters (copied from https://github.com/srstevenson/dotfiles)
* HighlightLine
* HighlightWord
* HighlightColumns
* ClearHightlights
